Este repositório reúne meu portfólio pessoal, onde compartilho meus projetos, serviços e experiências profissionais na área de desenvolvimento Backend. Programo por paixão à tecnologia e estou sempre em busca de novos desafios.

### Backend
- **Node.js** - Desenvolvimento de APIs escaláveis e microsserviços
- **Python/Django** - Aplicações web robustas e sistemas complexos
- **APIs REST** - Design e implementação RESTful
- **Banco de Dados** - PostgreSQL, MySQL, MongoDB, Redis

### DevOps
- **Docker** - Containerização de aplicações
- **CI/CD** - Automação com GitHub Actions e Jenkins
- **Git & GitHub** - Controle de versão e colaboração

### Frontend (Este Portfólio)
- **HTML5** - Estrutura semântica
- **CSS3** - Design responsivo e moderno
- **JavaScript Puro** - Interatividade sem frameworks

## 🛠️ Como Usar

### Visualizar Localmente

1. Clone o repositório:
```bash
git clone https://github.com/Danilo019/My_Portifolio_Danilo.git
cd My_Portifolio_Danilo
```

2. Abra o arquivo `index.html` em seu navegador:
```bash
# Usando Python
python -m http.server 8000

# Usando Node.js (http-server)
npx http-server

# Ou simplesmente abra o arquivo index.html no navegador
```

3. Acesse `http://localhost:8000` no navegador

### Personalizar

Para personalizar o portfólio para seu uso:

1. **Informações Pessoais** - Edite o `index.html`:
   - Nome e título na seção Hero
   - Descrição na seção Sobre
   - Informações de contato

2. **Projetos** - Modifique os cards de projeto:
   - Títulos e descrições
   - Links para repositórios
   - Tags de tecnologias

3. **Estilos** - Personalize em `styles/main.css`:
   - Cores (variáveis CSS no início do arquivo)
   - Fontes e tipografia
   - Espaçamentos

4. **Funcionalidades** - Estenda em `scripts/main.js`:
   - Adicione novos componentes
   - Integre com APIs
   - Implemente analytics
